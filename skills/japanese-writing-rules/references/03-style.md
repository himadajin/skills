# 文章の体裁の方針

文章の表現と体裁に関する方針を示す。

## 太字による強調の禁止

文章中で Markdown 記法の太字(`**`)を使って強調してはならない。
太字は読み手の環境によって表示が異なるため、認識のズレにつながる。
また、太字を使わないと情報の重要度が伝わりにくくなる文章は書くべきではない。

ただし、以下のように節の見出しや区切りを示す印として定型的に使う場合に限り例外的に認める。この用法も推奨はしない。

**LABEL**

## 冗長な語句の削除

以下の語句が含まれている場合、削除して文意が通るかを確認する。

- 「としては」
- 「においては」
- 「に対しては」
- 「については」
- 「という」

文意が通るなら削除する。文意が通らない場合は、語句を残すのではなく、文の構造を見直して語句なしで意味が通る文に書き直す。
ただし「Aとしては正しいがBとしては誤りである」のように明示的な対比構文で使う場合は残してよい。

## 曖昧な範囲表現の回避

以下の語で範囲を曖昧にしないこと。

- 「など」
- 「等」

「AやBなど」と書いた場合、読み手は「など」に何が含まれるかを判断できない。
列挙が網羅的であれば「など」を外す。
網羅できない場合は、列挙した項目の上位概念で範囲を示すか、条件を明記して範囲を限定する。

## 曖昧な推量表現の回避

以下の推量表現は避けること。

- 「思われる」
- 「考えられる」
- 「可能性がある」

事実であれば断定し、不確かであれば根拠や条件を明示する。

## 文体の統一

同一の文書内で文体を混在させないこと。
